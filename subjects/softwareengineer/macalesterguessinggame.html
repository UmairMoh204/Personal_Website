The Challenge

As a student at Macalester College, I wanted to create an engaging web application that would help students and visitors learn more about the campus while having fun. The challenge was to build an interactive guessing game that would test users' knowledge about Macalester College facts, history, and campus life.

I wanted to create something that would be both educational and entertaining, encouraging users to explore and learn more about the college community. The game needed to be intuitive, responsive, and provide immediate feedback to keep users engaged.

Development Process

Phase 1: Game Design & Planning

I started by researching interesting facts about Macalester College and designing the game mechanics. I created a comprehensive database of questions covering various topics like campus history, student life, academic programs, and fun trivia.

Phase 2: Core Game Logic

I implemented the core game functionality using JavaScript, including question randomization, score tracking, and user input validation. The game needed to handle different types of questions and provide appropriate feedback.

Phase 3: User Interface & Experience

I designed an intuitive and visually appealing interface using HTML5 and CSS3. The design focused on creating a welcoming atmosphere that reflects Macalester's community spirit while maintaining excellent usability.

Technical Challenges & Solutions

Challenge 1: Question Randomization

I needed to ensure questions appeared in random order while preventing repetition. I solved this by implementing a shuffle algorithm and maintaining a question history to avoid duplicates.

Challenge 2: Score Persistence

Users wanted their scores to be saved between sessions. I implemented localStorage to persist high scores and game progress, creating a more engaging experience.

Challenge 3: Responsive Design

The game needed to work seamlessly across different devices and screen sizes. I used CSS Grid and Flexbox to create a responsive layout that adapts to various viewport sizes.
